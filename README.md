** AI-Based Facial Recognition Attendance Management System**

Description

This project is an AI-powered attendance management system that uses facial recognition to automatically identify individuals and record attendance. 
It eliminates manual attendance, improves accuracy, and provides secure, real-time attendance tracking.

## Features
- Face detection and recognition
- Automatic attendance recording
- Real-time identification
- Improved accuracy and security

## Technologies Used
- Python
- OpenCV
- NumPy
- Face Recognition
- SQL database
